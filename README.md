# c-sim README

## Requirements

- SDL2
- SDL2_image
- SDL2_ttf

## Installation

### Windows

1. Clone the repository:
   ```bash
   git clone https://github.com/mjfwebb/c-sim.git
   ```
2. Build the project:
   ```bash
   ./build.bat
   ```
3. Run the executable:
   ```bash
   ./cultivation.exe
   ```

### Linux

1. Clone the repository:

   ```bash
   git clone https://github.com/mjfwebb/c-sim.git
   ```

2. Install dependencies:

   ```bash
   sudo apt install libsdl2-dev libsdl2-image-dev libsdl2-ttf-dev
   ```

3. Build the project:
   ```bash
   ./build.sh
   ```
4. Run the compiled binary:
   ```bash
   ./cultivation
   ```
